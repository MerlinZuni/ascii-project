## Video ASCII Feed

An experimental creative coding project that transforms live webcam input into abstract, ASCII-style visualizations in real time.

Built using **p5.js**, **JavaScript**, and **HTML**, this project explores the intersection of video, code, and generative aesthetics.

---

## Live Demo

👉 https://username.github.io/ascii-project

> ⚠️ **Important:** You must allow **webcam access** in your browser to experience the live visualization.

---

## Concept

This project converts live video input into dynamic ASCII representations, generating a constantly evolving visual output based on light, contrast, and motion.

It was created as part of an exploration into:
- Generative art  
- Real-time media processing  
- Creative coding techniques  

---

## Tech Stack

- **p5.js** – for rendering and video processing  
- **JavaScript** – core logic and transformations  
- **HTML/CSS** – structure and presentation  

---

## Context

This project was developed during studies in **Creative Coding at ZHdK (Zurich University of the Arts)**.

It serves as an experiment in blending:
- Visual art  
- Interactive media  
- Code-driven aesthetics - Think Matrix
